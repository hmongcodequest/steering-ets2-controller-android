# Steering Controller - Android Client (APK)

This repository hosts the compiled Android application (`.apk`) for the V-Joy Steering Controller project.

## 🚀 Download & Installation

1. Download the latest `steering-ets2-v1.0.0.apk` directly from this repository.
2. Transfer the file to your Android phone (or download it directly on your phone).
3. Open the `.apk` file and tap **Install**. (You may need to enable "Install from Unknown Sources" in your Android settings).
4. Run the app and connect to your PC Receiver via Wi-Fi or USB.

## 🔧 Features

- Real-time steering wheel emulation for Euro Truck Simulator 2 (ETS2).
- Wi-Fi (Wireless) and USB (Wired via ADB reverse) connection modes.
- Secure Challenge-Response authentication.
- Built-in buttons for Horn, Turn Signals, Headlights, Hazard Lights, and Wipers.
